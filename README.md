# 撮影する環境を作成する 
まず、ワークスペースを作成する。shootは例
```
mkdir shoot
cd shoot
mkdir src
cd src
milab@elvis:/mnt/test/shoot/src$ pwd
/mnt/test/shoot/src

catkin_init_workspace
cd ../
catkin_make
source devel/setup.bash
```
次にmodel_gazeboを/shoot/src/に配置する
